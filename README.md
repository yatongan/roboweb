# RoboLife - Professional 3D Measurement & Reconstruction Website

A modern, responsive website for RoboLife, a company specializing in high-quality 3D measurement and reconstruction solutions, including both hardware development and software development.

## 🌟 Features

- **Professional Design**: Modern, clean interface with gradient backgrounds and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Multiple Pages**:
  - Home (index.html) - Main landing page with service overview
  - Services (services.html) - Detailed service offerings
  - Hardware (hardware.html) - 3D scanning equipment and specifications
  - Software (software.html) - Software solutions and tools
  - About (about.html) - Company information and team
  - Contact (contact.html) - Contact form and office locations
- **Interactive Elements**: Smooth scrolling, form validation, animations on scroll
- **Brand Colors**: Professional blue (#0066cc) and cyan (#00a3e0) color scheme

## 🎨 Customization

### Changing Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00a3e0;
    --dark-color: #1a1a1a;
    --light-color: #f5f5f5;
    --text-color: #333;
}
```

### Updating Content
- Edit HTML content directly in each `.html` file
- Update company information in `about.html`
- Modify services in `services.html`
- Update contact details in `contact.html`

### Adding New Pages
1. Create a new `.html` file
2. Copy the basic structure from existing pages
3. Add link to navigation bar in all files
4. Style as needed using existing CSS classes
